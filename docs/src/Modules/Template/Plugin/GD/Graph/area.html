The GD.Graph.area plugin provides an interface to the GD::Graph::area class defined by the GD::Graph module. It allows one or more (x,y) data sets to be plotted as lines with the area between the line and x-axis shaded, in addition to axes and legends.

See [% ttlink('GD::Graph') -%] for more details.
